MAST

AI-powered accountability with real consequences

MAST is a self-improvement tool that uses accountability deposits to help people follow through on their goals. An AI agent manages your commitments, tracks your progress, and returns your deposit when you succeed. Built on Base with USDC.

How It Works

MAST is an accountability tool powered by an AI agent (via the Model Context Protocol). The agent acts as a personal accountability partner — it helps users define goals, place accountability deposits on those goals, and verifies completion.

Users fund their account once with a credit card. The agent manages individual commitments from that balance. Completed goals return the deposit. Missed deadlines mean the deposit is not returned.

1
Setup
User connects to the agent. The agent calls mast_setup and learns about the user — their goals, what drives them, their preferences.
2
Fund
User adds funds once via credit card through Coinbase Onramp. The payment appears as a normal card charge. USDC is deposited into a secure accountability contract on Base. Zero fees.
3
Commit
User tells the agent what they'll do. The agent creates a commitment with a personalized message, deposit amount, deadline, and strictness level. Funds are locked from the pre-funded balance.
4
Complete or Learn
User reports completion to the agent → deposit returns to available balance. Deadline passes without completion → deposit is not returned, reinforcing the habit.

Features

Recurring Commitments

Daily, weekly, or one-off. The agent manages the cadence automatically. Funds lock and release each period.

Strictness Levels

From iron (no cancellation, no excuses) to chill (cancel anytime). Match the deposits to the goal.

Personalized Pages

Every commitment gets a unique page with the user's name, colors, mantra, and a message from the agent.

Smart Contract Escrow

Funds are held in an auditable accountability contract on Base. Transparent. Verifiable. No trust required.

Zero-Fee Funding

Credit card payments via Coinbase Onramp convert to USDC on Base with zero fees. Users see dollars, not crypto.

Pre-Funded Balance

Fund once, commit many times. No per-task payment friction. Balance carries across all active commitments.

Why MAST Exists

Research in behavioral economics shows that financial commitment devices dramatically increase follow-through rates. People who put money on a goal are 2–3x more likely to achieve it.

  • Loss aversion is the strongest motivator — stronger than rewards, reminders, or willpower
  • AI agents can now manage goals autonomously, but they lack real-world consequence mechanisms
  • MAST gives agents the ability to create binding, verifiable, financial commitments
  • The smart contract makes deposits transparent — neither the user nor the platform can manipulate funds

Payment Flow

MAST uses Coinbase Onramp as the payment gateway. Users pay with their credit card, debit card, Apple Pay, or Google Pay. The payment is automatically converted to USDC and deposited into a secure smart contract on Base — all behind the scenes.

Users never interact with crypto directly. They see dollar amounts, pay with familiar methods, and receive refunds to their available balance on task completion. The blockchain is infrastructure, not interface.

Why Base + USDC: Near-zero gas fees (<$0.01 per transaction), instant finality, and Coinbase Onramp provides zero-fee conversion from fiat to USDC on Base. This means a $1 commitment actually locks $1 — no fees eaten in transit.

Technical Stack

MCP Server (Node.js) Base L2 USDC Solidity (Escrow Contract) Coinbase Onramp Coinbase CDP ethers.js

The MCP server integrates with any AI agent that supports the Model Context Protocol — including Claude, and any MCP-compatible assistant. The server manages wallets, commitments, accountability deposits, and payment page generation.

Security

MAST is built with security as a first principle:

  • Authenticated API — Session tokens are generated server-side via an authenticated endpoint. API keys are never exposed to clients.
  • CORS protection — Backend API restricts access to approved origins only. No wildcard access.
  • No stored payment data — All payment processing is handled by Coinbase. MAST never sees or stores credit card details.
  • Auditable smart contract — All fund movements are recorded on-chain and publicly verifiable on Base.
  • User-controlled withdrawals — Users can withdraw their available balance back to fiat at any time via Coinbase Offramp.

Use Cases

MAST is designed for anyone who wants to follow through on their commitments with real financial accountability:

  • Daily habits — meditation, exercise, journaling, reading
  • Productivity — deep work sessions, project deadlines, shipping features
  • Health — gym attendance, meal prep, sleep schedules
  • Learning — language practice, course completion, skill development
  • Creative work — writing daily, practicing an instrument, drawing
